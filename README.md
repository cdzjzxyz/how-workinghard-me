# 企业考勤管理系统

## 项目介绍
这是一个基于Spring Boot + Vue.js开发的企业考勤管理系统，旨在提供完整的员工考勤管理解决方案。系统支持员工请假申请、考勤记录、公告管理、通知提醒等功能，帮助企业实现高效的考勤管理。系统采用前后端分离架构，提供友好的用户界面和完善的后台管理功能。

## 系统特点
1. **智能考勤管理**
   - 自动判断考勤状态（正常、迟到、早退、缺卡）
   - 支持多种请假类型（事假、病假、年假等）
   - 考勤数据实时统计和可视化展示
   - 考勤异常自动提醒

2. **绩效评分系统**
   - 自动计算考勤绩效分数
   - 支持自定义评分规则
   - 绩效数据导出Excel
   - 绩效趋势分析图表

3. **人性化通知系统**
   - 考勤异常实时通知
   - 请假审批结果推送
   - 公告发布提醒
   - 心声评论互动通知

4. **员工心声平台**
   - 匿名发布心声
   - 评论互动功能
   - 内容审核机制
   - 情感分析统计

5. **数据可视化**
   - 考勤统计图表
   - 部门考勤对比
   - 请假类型分布
   - 绩效分数趋势

## 技术栈
### 后端
- Spring Boot 2.x：提供快速开发框架
- MyBatis：持久层框架，实现数据库操作
- MySQL：关系型数据库，存储业务数据
- Spring Security：安全框架，处理认证和授权
- JWT认证：实现无状态的用户认证
- Lombok：简化Java代码
- POI：处理Excel导入导出

### 前端
- Vue.js 3：渐进式JavaScript框架
- Element Plus：基于Vue 3的UI组件库
- Axios：处理HTTP请求
- Vue Router：实现前端路由
- Vuex：状态管理
- ECharts：数据可视化图表
- Day.js：日期时间处理

## 安装说明

### 后端部署
1. 环境要求
   - JDK 1.8+
   - Maven 3.x
   - MySQL 5.7+
   - Redis（可选，用于缓存）

2. 配置数据库
   - 创建数据库：zxyf
   - 导入SQL脚本（sql表和触发器.txt）
   - 配置数据库连接信息

3. 修改配置
   - 修改application.yml中的数据库连接信息
   - 配置JWT密钥
   - 配置文件上传路径
   - 配置跨域设置

4. 启动服务
```bash
cd kaoqin
mvn spring-boot:run
```

### 前端部署
1. 环境要求
   - Node.js 14+
   - npm 6+
   - Vue CLI 4+

2. 安装依赖
```bash
cd test_vue
npm install
```

3. 开发环境运行
```bash
npm run serve
```

4. 生产环境部署
```bash
npm run build
```
